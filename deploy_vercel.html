AIQToolkit NVIDIA UI - Vercel Deployment

Quick Deploy

Deploy AIQToolkit NVIDIA UI to Vercel with a single click:

Deploy to Vercel

Important: Add this environment variable during deployment:

NVIDIA_ENDPOINT = https://jupyter0-s1ondnjfx.brevlab.com

1 Create a GitHub Repository

First, you need to create a GitHub repository to host your code.

  1. Go to https://github.com/new
  2. Enter a repository name (e.g., aiqtoolkit-nvidia-ui)
  3. Set the repository to Public or Private (Public is recommended for easier Vercel deployment)
  4. Click "Create repository"
Create GitHub Repository

2 Push Your Code to GitHub

Next, push your AIQToolkit NVIDIA UI code to GitHub.

git init
git add .
git commit -m "Initial commit: AIQToolkit NVIDIA UI"
git remote add origin https://github.com/YOUR_USERNAME/aiqtoolkit-nvidia-ui.git
git push -u origin main

Replace YOUR_USERNAME with your GitHub username.

Note: If you're not in the web-ui directory, navigate to it first with:

cd /path/to/AIQToolkit/web-ui

3 Deploy to Vercel

Now, deploy your code to Vercel:

  1. Go to https://vercel.com/new
  2. Click "Import Git Repository"
  3. Select your GitHub repository: YOUR_USERNAME/aiqtoolkit-nvidia-ui
  4. Configure your project:
    • Framework Preset: Other
    • Root Directory: ./
    • Build Command: (leave empty)
    • Output Directory: ./
  5. Add Environment Variables:
    • Name: NVIDIA_ENDPOINT
    • Value: https://jupyter0-s1ondnjfx.brevlab.com
  6. Click "Deploy"
Deploy to Vercel

4 Access Your Deployed Application

After deployment is complete, you can access your application at:

https://aiqtoolkit-nvidia-ui.vercel.app

(Your URL may be different based on your repository name)

Note: It may take a few minutes for your deployment to complete and become available.

5 Verify Functionality

Once deployed, verify these key features:

  • Main Dashboard: https://your-app.vercel.app/
  • Control Tower: https://your-app.vercel.app/public/control-tower.html
  • Reasoning Interface: https://your-app.vercel.app/public/reasoning.html
  • API Health Check: https://your-app.vercel.app/api/health

Important: If any features don't work, check your environment variables and Vercel deployment logs.
