Setup GitHub Repository for AIQToolkit NVIDIA Blueprint

Follow these steps to create a GitHub repository for the AIQToolkit NVIDIA Blueprint that will work with the launchable.

Step 1: Create a new GitHub repository

Create a public repository named aiqtoolkit-nvidia.

Create New Repository

Use these settings:

  • Repository name: aiqtoolkit-nvidia
  • Description: AIQToolkit NVIDIA Blueprint - A web interface for monitoring NVIDIA AI services
  • Visibility: Public (important!)
  • Initialize this repository with a README: ✓

Important: The repository must be public for the launchable to access it.

Step 2: Clone the repository

After creating the repository, clone it to your local machine:

git clone https://github.com/YOUR_USERNAME/aiqtoolkit-nvidia.git
cd aiqtoolkit-nvidia

Replace YOUR_USERNAME with your GitHub username.

Step 3: Copy files to the repository

Copy all files from the web-ui directory to your cloned repository:

cp -R /Users/apple/projects/AIQToolkit/web-ui/* /path/to/cloned/aiqtoolkit-nvidia/

Replace /path/to/cloned/aiqtoolkit-nvidia/ with your actual cloned repository path.

Step 4: Commit and push the files

git add .
git commit -m "Add AIQToolkit NVIDIA Blueprint files"
git push

Step 5: Create the launchable

Now return to the launchable creation page and use:

  • GitHub Repository: https://github.com/YOUR_USERNAME/aiqtoolkit-nvidia
  • Runtime Environment: VM Mode w/ Jupyter
  • GPU: NVIDIA L40S
  • File to Run: launch.ipynb

Alternative: Direct Upload

If you prefer not to use git commands, you can also upload the files directly through GitHub's web interface:

  1. After creating the repository, click on "Add file" > "Upload files"
  2. Drag and drop all files from the web-ui directory
  3. Click "Commit changes"

Note: If you have many files, you may need to upload them in batches.
